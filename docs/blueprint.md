# **App Name**: Concurso Sprint

## Core Features:

- Question Bank: Browse and answer a wide range of questions categorized by subject and difficulty level.
- Simulated Exams: Generate realistic exam simulations based on chosen subjects and difficulty to assess readiness.
- Flashcard System: Create, study, and manage flashcards for efficient memorization of key concepts.
- Study Time Tracker: Automatically track total study time, providing insights into study habits and progress.
- Performance Analytics: Generate comprehensive statistics on performance across subjects and question types, highlighting strengths and weaknesses.
- Daily Login Streak: Maintain a consecutive day login streak to earn bonuses and track commitment.
- Personalized AI Study Plan: Using generative AI, the app can generate a tailored study plan based on the user's performance and goals. The AI uses its tool to access performance stats, recommended resources, and available time.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and intelligence.
- Background color: Light gray (#ECEFF1) for a clean and focused learning environment.
- Accent color: Vibrant orange (#FF9800) to highlight important actions and notifications.
- Body and headline font: 'Inter', a grotesque sans-serif font that looks modern and neutral, for a machined feel.
- Use clear, professional icons for each menu item and function.
- Prioritize a clean, intuitive layout with clear sections for each feature to minimize distraction.
- Use subtle animations to provide feedback and guide the user through the application.
